## 3. 핵심 재무 요약표 ★ (가장 중요)

**재무 선배 스타일의 핵심**: 매출액, 영업이익, 당기순이익 등 **절대 숫자**를 연도별 표로 먼저 보여준다.

### 3-1. 실적 요약 (연도별 표)

| 항목 | 2023년 | 2024년 | 2025년 | 전년 대비 |
|------|--------|--------|--------|----------|
| **매출액** | 258.9조 | 300.9조 | 333.6조 | +10.9% |
| **매출원가** | - | - | - | - |
| **매출총이익** | - | - | - | - |
| **영업이익** | - | - | - | - |
| **당기순이익** | - | - | - | - |
| **자산총계** | - | 514.5조 | 566.9조 | +10.2% |
| **부채총계** | - | 112.3조 | 130.6조 | +16.3% |
| **자본총계** | - | 402.2조 | 436.3조 | +8.5% |

**왜 이 표가 먼저인가:**
- 재무 선배 분석에서 항상 5개년 실적표가 가장 먼저 나옴
- 숫자를 먼저 보여주고 → 해석은 그 아래에서
- 사용자가 "이 회사가 얼마를 벌었는지"를 한눈에 파악

### 3-2. 실적 추이 차트

- **막대 그래프**: 매출액 연도별 추이 (가장 직관적)
- **라인 겹침**: 영업이익, 순이익을 매출 위에 라인으로 겹쳐 표시
- **이중축**: 좌축=금액(조원), 우축=이익률(%)

## 4. 카테고리별 상세 분석

### 4-1. 사업모델 분석

**구성:**
- 매출원가율 vs 판관비율 비율 분석 (재무 선배 스타일)
- 비용 구조 파이 차트 또는 스택 바
- AI 분석 코멘트 (카드 형태)

**재무 선배 참고:**
- "매출원가율 82%면 식품이라 어쩔 수 없다"
- "판관비율 30%면 배송비가 큰 비중"
- 원가 구성을 분해해서 보여주는 것이 핵심

**표시 항목:**
| 항목 | 값 | 비고 |
|------|------|------|
| 매출원가율 | XX% | 매출원가 / 매출액 |
| 판관비율 | XX% | 판관비 / 매출액 |
| 영업비용률 | XX% | (원가+판관비) / 매출액 |

### 4-2. 매출·이익 흐름 ★

**이 섹션이 재무 선배 분석의 핵심.**

**구성:**
1. **매출 추이 막대 그래프** (연도별)
2. **이익 추이 라인 차트** (영업이익 + 순이익)
3. **마진율 추이** (영업이익률 + 순이익률 라인)
4. **성장률 표**:
   | 항목 | YoY 성장률 | 판단 |
   |------|-----------|------|
   | 매출 성장률 | +10.9% | 🟢 양호 |
   | 영업이익 성장률 | - | ⚪ |
   | 순이익 성장률 | - | ⚪ |

5. **AI 해설**: "양적 성장(매출)과 질적 성장(이익) 중 어느 쪽이 더 강한지" 판단
   - 재무 선배: "고성장에도 불구하고 이익은 개선되지 않을 것"

### 4-3. 재무건전성

**구성:**
1. **안정성 지표 KPI 카드** (큰 숫자 + 상태 배지)
   - 부채비율: XX%
   - 자기자본비율: XX%
   - 유동비율: XX%
   - 이자보상배율: XX배

2. **부채 vs 자본 스택바 차트** (연도별)

3. **현금 현황**:
   | 항목 | 값 |
   |------|------|
   | 현금·현금성자산 | XX억 |
   | 영업활동 현금흐름 | XX억 |

4. **AI 해설**: "이 회사가 갑자기 어려워지면 버틸 수 있는가?"

### 4-4. 리스크 신호

**구성:**
1. **경고 항목 체크리스트** (체크박스 형태)
   - [ ] 자본잠식 징후
   - [ ] 부채 급증 (전년 대비 20% 이상)
   - [ ] 영업이익 적자 전환
   - [ ] 이자보상배율 1배 미만
   - [ ] 현금흐름 적자

2. **급변동 지표 하이라이트**:
   전년 대비 ±30% 이상 변동한 항목을 빨간색으로 강조

3. **AI 해설**: "초보자가 이 숫자를 보면 왜 위험한 건지" 설명

### 4-5. 비교·평가

**구성:**
1. **효율성 KPI 카드**
   - ROE: XX% (업종 평균 대비)
   - ROA: XX%

2. **투자 매력도** (별 5개 또는 숫자 점수)

3. **주목 포인트** (2~3가지 불릿)

4. **AI 해설**: 종합 평가

## 구현 시 핵심 원칙

### 1. "숫자 먼저, 해석 나중" (재무 선배 원칙)
- 절대 숫자(매출액, 영업이익 등)를 표로 먼저 보여줌
- AI 해설은 그 아래에 카드 형태로 배치
- 숫자가 텍스트에 묻히면 안 됨

### 2. "연도별 비교가 기본" (시계열 분석)
- 최소 2개년, 가능하면 3~5개년 비교
- 전년 대비 증감률(↑↓) 표시 필수
- 추세를 보여주는 라인 차트

### 3. "카테고리별 분리, 각각에 AI 코멘트"
- 5개 카테고리를 별도 Card로 분리
- 각 카드 안에: 숫자 표/차트 + AI 해설 박스
- 전체를 한 테이블에 우겨넣지 않음

### 4. "색상 신호등" (직관적 판단)
- 🟢 양호: 업종 평균 이상 / 건전한 수준
- 🟡 보통: 평균 수준 / 관찰 필요
- 🔴 주의: 업종 평균 이하 / 리스크

### 5. "큰 숫자 + 작은 설명" (KPI 카드 패턴)
```
┌─────────────┐
│ 영업이익률   │  ← 작은 라벨
│   15.3%     │  ← 큰 숫자 (2xl font)
│ 🟢 양호     │  ← 상태 배지
│ 전년 +2.1%p │  ← 변화량
└─────────────┘
```

## 현재 코드 대비 필요한 변경

### API 변경 (/api/explain)
- `ChecklistResult`에 `financialSummary` 추가 (연도별 핵심 숫자표 데이터)
- 각 checklist item의 `analysis`를 더 구체적으로

### 프론트엔드 변경 (ResultView.tsx)
1. **섹션 3 추가**: 핵심 재무 요약표 (metricsPerYear + raw financial data)
2. **체크리스트 표 → 카테고리별 카드 분리**: 각 카드 안에 숫자 + AI 해설
3. **차트 통합**: 각 카테고리 카드 안에 해당 차트 포함
4. **KPI 카드 강화**: 전년 대비 변화량 표시

### AnalysisPanel 변경
- `FinancialData` (원시 데이터)도 ResultView에 전달해야 함
- 현재는 metricsPerYear(계산된 비율)만 전달 → raw 숫자(매출액 등)가 필요

## 재무 선배 분석 스타일 요약

| 특징 | 설명 |
|------|------|
| **톱다운** | 큰 그림(실적 요약) → 세부(비용 구조) → 결론(수익화 방법) |
| **비교** | 연도별 + 동종업계 + 비즈니스 모델 간 비교 |
| **수식 제시** | 지표 계산 공식을 보여줘서 이해도 높임 |
| **비유 활용** | "원가율 82%면 100원 팔아서 82원은 원가" |
| **구조 분해** | 비용을 변동비/고정비로 나누고 BEP 분석 |
| **정성+정량** | 숫자 분석 + 전략적 의미 해석 병행 |
